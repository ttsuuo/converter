# 💱 Currency Converter with Exchange Rate Chart

[![Live Demo](https://ttsuuo.github.io/converter/)](https://ttsuuo.github.io/converter/)  

## 📖 Description

A real-time currency converter web application with historical exchange rate visualization. Features light/dark theme switching and a persistent conversion history stored in LocalStorage.

**Key Features:**
- Convert between 4 currencies (USD, EUR, RUB, BYN)
- Interactive chart showing exchange rate trends over the last month (Chart.js)
- Conversion history: saves last 5 operations in LocalStorage
- Dark/Light theme toggle with saved preference
- Fully responsive: works on desktop, tablet, and mobile

## 🛠️ Tech Stack

- **JavaScript (ES6+)** — application logic
- **Chart.js** — chart rendering
- **LocalStorage** — history & theme persistence
- **REST API (Frankfurter)** — real-time and historical exchange rates
- **HTML5, CSS3, BEM** — structure and styling
- **Git & GitHub** — version control

## 📂 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ttsuuo/converter.git
   ```
2. Open `index.html` in your browser.

No dependencies or build tools required — just open and use.
